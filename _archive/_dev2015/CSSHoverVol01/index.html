CSS Hover Effects Vol. 01

I love CSS hover effects with transitions. They can really make a website come alive and draw the users attention. It can also increase conversion simply by making a call-to-action wiggle and flash, like deep sea fish. However, be careful when developing for mobile devices. Tablets and smaller touch devices are not hover-friendly. I like to use SASS breakpoints to ensure only my hover effects run on larger screens where the user would be using a mouse. It would be far more failproof to actually detect if the user is on a touch screen device. Since we remove the hover effects on mobile, it's important to note that you should be displaying any hidden content that is animating on hover.
